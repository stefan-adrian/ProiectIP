ProiectIP - Ultimate Blackjack

Pentru a deschide proiectul aveti nevoide de Visual Studio 2013(sau de o versiune asemanatoare care sa il compileze.

Puteti dechide jocul prin executabilul din locatia Blackjack/Debug/Blackjack.exe.

De asemena puteti deschide proiectul din locatia Blackjack/Blackjack.sln si sa executati proiectul pentru a porni aplicatia/
